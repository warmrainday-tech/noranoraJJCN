# 绫月奶萝 Ayatsuki Nora 粉丝站

🎨 虚拟主播兼画师绫月奶萝的官方粉丝网站

## 功能

- 🏠 首页 - 奶萝简介和社交链接
- 🎭 人设 - 角色设定和背景资料
- 🎨 插画作品展示
- 🎬 切片视频回顾
- 🎮 互动小游戏

## 技术栈

- 纯 HTML + CSS + JavaScript
- 响应式设计
- 无需后端，静态部署

## 本地预览

```bash
# 方式1: Python
python -m http.server 8000

# 方式2: PHP
php -S localhost:8000

# 方式3: Node.js
npx serve .
```

然后访问 http://localhost:8000
